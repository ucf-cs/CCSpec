About CCSpec

Contributing to Tervel

If you wish to contribute please send an email to tervel@cse.eecs.ucf.edu. If we don't respond, hunt us down on github or linkedin.

How can I contribute to Tervel?

We are looking for C++ application designers willing to tryout the algorithms and containers implemented in Tervel. If you are such an adventurer, then send us any and all bug reports, feature request, compliments, complaints, funny jokes, etc.
