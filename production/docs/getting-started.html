Installing CCSpec

System Requirements

Supported platforms:

  • Ubuntu Linux

Downloading CCSpec

Clone CCSpec from the git repo:

$ git clone https://github.com/ucf-cs/CCSpec

Building CCSpec

Dependencies

  • Flex

  • Bison

Building

$ cd CCSpec
$ make
$ make experiments

Running CCSpec Experiments

$ cd CCSpec
$ ./run.sh experiments/IQueue/test.o
